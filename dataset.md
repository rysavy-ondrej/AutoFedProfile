# Dataset

##  Define Dataset Scope and Taxonomy

| Level | Focus | Label Granularity | Typical Classes | Purpose |
|-------|--------|------------------|-----------------|----------|
| **L1: Device/OS Identification** | Network identity & behavior | Device type, OS version | Windows 10/11, Android, iOS, routers, IoT cameras, smart TVs | Low-level fingerprinting |
| **L2: User/Application Profiling** | Application and behavioral context | App name, user behavior class (e.g. office, streaming, gaming) | Browser, Teams, Zoom, Steam, Office, etc. | Usage and behavior modeling |
| **L3: Malware Family Characterization** | Malicious communication | Malware family, C2 technique, encryption pattern | Trickbot, Emotet, Zeus, etc. | Threat intelligence and separation from benign |
